Função Segundo Grau

Calculadora Aprender

O que é uma Função de Segundo Grau?

Uma função de segundo grau, também chamada de **função quadrática**, é representada pela fórmula:

f(x) = ax² + bx + c

Onde:

  • a é o coeficiente quadrático (nunca pode ser 0);
  • b é o coeficiente linear;
  • c é o termo constante.

Características da Função de Segundo Grau

Algumas das principais características incluem:

  • O gráfico da função é uma **parábola**, que pode estar voltada para cima (se a > 0) ou para baixo (se a < 0).
  • O vértice da parábola é o ponto onde ela muda de direção, calculado por: xv = -b / (2a) e yv = f(xv).
  • As raízes da função (se existirem) são os valores de x para os quais f(x) = 0.

Como Resolver uma Função de Segundo Grau?

Para encontrar as raízes de uma função, utilizamos a fórmula de Bhaskara:

x = (-b ± √Δ) / 2a

Onde Δ (delta) é calculado como:

Δ = b² - 4ac

Dependendo do valor de Δ, temos:

  • Se Δ > 0: duas raízes reais distintas;
  • Se Δ = 0: uma raiz real (as duas raízes são iguais);
  • Se Δ < 0: nenhuma raiz real.

Exemplo Prático

Considere a função:

f(x) = 2x² - 4x - 6

Passo a passo:

  • Coeficientes: a = 2, b = -4, c = -6;
  • Calculando Δ: Δ = (-4)² - 4(2)(-6) = 16 + 48 = 64;
  • Calculando as raízes: x1 = (-(-4) + √64) / (2*2) = 3, x2 = (-(-4) - √64) / (2*2) = -1.

O gráfico dessa função será uma parábola voltada para cima, com vértice em (xv, yv).
